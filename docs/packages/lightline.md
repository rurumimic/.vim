# Lightline

- github: [itchyny/lightline.vim](https://github.com/itchyny/lightline.vim)

## Add Plugin

```vim
call plug#begin()

Plug 'itchyny/lightline.vim'

call plug#end()
```

## Statusline

- :help [statusline](https://vimhelp.org/options.txt.html#%27statusline%27)

current statusline:

```vim
:set statusline?
:echo &statusline
```

set statusline:

```vim
:set statusline=%f
```
